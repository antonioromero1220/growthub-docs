# Growthub Documentation - Custom Image Generation Prompts

These are the exact image generation prompts for each core feature page hero image. Use these prompts with your preferred AI image generation tool (Midjourney, DALL-E, Stable Diffusion, etc.)

## Brand Guidelines for All Images
- **Primary Colors**: Cyan (#5bbfcf) to Blue (#1a6b9c) gradient
- **Style**: Modern tech, futuristic, professional SaaS aesthetic
- **Mood**: Innovative, powerful, clean
- **Format**: Wide aspect ratio (approximately 2.8:1 for hero sections)

## Post-Processing Recommendations

1. Apply cyan-to-blue gradient overlay at 30-40% opacity
2. Add subtle vignette for depth
3. Ensure dark areas are true black (#000 or near) for contrast
4. Consider adding subtle grain texture for premium feel
